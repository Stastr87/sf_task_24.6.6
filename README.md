Ресурс для задания 24.6.6 (HW-02)
Задача:
Написать функцию позволяющую считывать из консоли адрес который необходимо обработать при помощи html анализатора.
Условия: Функция strings.NewReader возвращает знаение, соответвующее интерфейсу io.Reader, путем чтения своего аргумента (тип строка).
Необходимо реализовать простую версию NewReader и использовать ее для синтаксического анализатора HTML, код которого дан в репозитории.
Решение задачи должно быть добавлено в репозиторий в отдельной ветке.
