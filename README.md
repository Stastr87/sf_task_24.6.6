## Ресурс для задания 24.6.6 (HW-02)

### Задача:


Написать функцию, позволяющую считывать из консоли адрес, который необходимо обработать при помощи HTML‑анализатора.

Условия: функция strings.NewReader возвращает значение, соответствующее интерфейсу io.Reader, путём чтения своего аргумента (тип — строка).

Необходимо реализовать простую версию NewReader и использовать её для синтаксического анализатора HTML, код которого дан в репозитории.

Решение задачи должно быть добавлено в репозиторий в отдельной ветке.
